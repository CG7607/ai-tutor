## 1. 快速开始

### 环境要求
- Python 3.11+
- 网络（访问 LLM API）

### 方式一：一键安装（推荐）

| 系统 | 操作 |
|------|------|
| **macOS** | 双击 `aitutor/install.sh`，或在终端运行 `bash aitutor/install.sh` |
| **Windows** | 双击 `aitutor/install.bat` |

安装脚本自动完成五步：Python 环境检测 → 虚拟环境创建 → 依赖安装 → API Key 配置 → 桌面快捷方式生成。安装完成后双击桌面快捷方式即可启动。

### 方式二：手动安装

```bash
# ① 安装依赖
pip install -r aitutor/requirements.txt

# ② 配置 API Key
cp aitutor/.env.example aitutor/.env
# 编辑 .env，将 LLM_API_KEY 改为你的真实 Key

# ③ 启动
cd aitutor && bash run.sh
```

打开浏览器访问 **http://localhost:8501**

## 2. 账号系统

### 注册与登录

打开页面后，左侧边栏顶部即为登录区域：

- **注册**：输入用户名（3-20 字符）、密码（至少 4 位）、确认密码，点击注册
- **登录**：输入已注册的用户名和密码
- 密码使用 SHA-256 加密存储，明文不落地

### 数据独立

每个用户的数据存储在 `data/users/用户名/` 下：

```
data/users/张三/
├── profile.json          # 密码哈希
├── chat_history.json     # 聊天记录（已废弃，改用 conversations/）
├── quiz_history.json     # 答题历史
├── wrong_answers.json    # 错题本
└── conversations/        # 对话历史
    ├── a1b2c3d4.json     # 对话1
    └── e5f6g7h8.json     # 对话2
```

### 退出登录

点击侧边栏「退出登录」按钮即可。

## 3. 智能问答

### 三个 AI 导师

| 导师 | 专长 | 触发条件 |
|------|------|---------|
| 数学推导 | 公式推导、数学原理 | 问题含公式/推导/概率/矩阵 |
| 算法实现 | 从零手写算法代码 | 问题含实现/手写/代码 |
| 概念辨析 | 对比易混淆概念 | 问题含区别/对比/vs |

Router 会自动分析问题并调度最合适的导师回答。

### 文件上传

输入框上方有文件上传区域，支持格式：txt、pdf、py、md、json、csv 等。上传后文件内容自动作为对话上下文，AI 导师会基于文件内容回答你的问题。

> PDF 解析需安装：`pip install pypdf`

### 对话管理

| 操作 | 说明 |
|------|------|
| 新建对话 | 侧边栏「＋ 新建对话」，自动保存当前对话 |
| 历史列表 | 侧边栏显示所有历史对话，● 标记当前活跃 |
| 切换对话 | 点击历史对话名即加载该对话继续聊 |
| 删除对话 | 点击对话旁的 🗑 按钮 |
| 自动新对话 | 每次登录自动创建新对话 |

每条回复后自动保存到磁盘，对话标题自动取第一条消息的前 40 字。

## 4. 知识图谱

### 图谱浏览

右侧展示完整的课程知识体系网络图，26 个节点、20 条关系。可拖拽、缩放。

### 点击交互

**直接点击图中节点**，弹出详情卡片，显示关联概念和关系。

### 深度解读

搜索概念后点击「查看关联」，左侧面板生成由 LLM 驱动的七维度深度解读：

| 维度 | 内容 |
|------|------|
| 核心思想 | 概念解决什么问题，最本质的思路 |
| 数学基础 | 核心公式（LaTeX）+ 直觉解释 |
| 历史背景 | 谁在哪年提出，如何演变 |
| 关键洞察 | 3 条精炼的深度理解 |
| 常见误区 | 学生最容易搞混的地方 |
| 应用场景 | 具体实例 |
| 延伸学习 | 推荐学习路径 |

> 深度解读需要 LLM 连接。图谱浏览和搜索无需 API Key。

### 实体类型

| 类型 | 颜色 | 示例 |
|------|------|------|
| 概念 | 🟢 绿 | 过拟合、正则化 |
| 算法 | 🔵 蓝 | K-means、CNN、Transformer |
| 人物 | 🟣 紫 | Hinton、LeCun |
| 前置知识 | 🟠 橙 | 线性代数、概率论 |

### 概念列表

| 分类 | 包含 |
|------|------|
| 数学基础 | 线性代数、概率论、微积分、Python |
| 机器学习 | 梯度下降、SGD、Adam、损失函数、交叉熵、过拟合、正则化、K-means、KNN、决策树、SVM、生成式模型、判别式模型 |
| 深度学习 | 反向传播、CNN、RNN、LSTM、Transformer、梯度消失、梯度爆炸 |
| 人物 | Hinton、LeCun |

## 5. 自适应测验

### 布鲁姆认知层次

| 层次 | 名称 | 能力 | 题型 |
|------|------|------|------|
| L1 | 记忆 | 回忆基本事实 | 单选、填空 |
| L2 | 理解 | 用自己的话解释 | 单选、多选、简答 |
| L3 | 应用 | 运用知识解决问题 | 选择、简答 |
| L4 | 分析 | 分解问题、识别关系 | 多选、简答 |
| L5 | 评价 | 基于标准做出判断 | 简答、开放 |
| L6 | 创造 | 综合知识设计方案 | 开放设计题 |

### 自适应调节

```
准确率 > 80% → 升级
准确率 < 50% → 降级
其他        → 保持
（答题数 < 3 时不调整）
```

### 知识点

梯度下降、反向传播、CNN、RNN/LSTM、Transformer、过拟合与正则化、损失函数、优化算法、生成式模型、K-means 聚类、SVM、决策树

每个知识点 6-8 个子维度，随机抽取，配合去重机制避免重复出题。

### 错题库（独立导航页）

答错的题自动存入错题库。通过侧边栏导航「错题库」或测验页面的「📌 进入错题库」按钮进入专页：

| 功能 | 说明 |
|------|------|
| 分页浏览 | 每页 10 题，左右翻页，支持海量错题 |
| 统计概览 | 错题总数、涉及知识点数一目了然 |
| 单条删除 | 每条右侧 🗑 按钮，删除后立即同步保存 |
| 清空全部 | 二次确认后方可清空，防止误操作 |
| 数据保护 | 原子写入 + 自动备份 `.bak`，防止数据意外丢失 |

> v2.1 起，错题库升级为独立导航页，不再占用测验页面空间，浏览体验更清爽。

### 做题流程

1. 选择知识点
2. 点击「生成题目」
3. 选择答案后点击「提交答案」
4. 查看对错和详细解析
5. 点击「生成下一题」继续

## 6. 配置说明

### 切换 LLM 服务商

修改 `aitutor/.env`：

```env
# DeepSeek（默认）
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat

# 通义千问
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL=qwen-plus

# OpenAI
LLM_BASE_URL=https://api.openai.com/v1
LLM_MODEL=gpt-4o
```

### 端口修改

启动时指定不同端口，并同步更新前端文件中的 `BACKEND_URL`。

## 7. 常见问题

**Q：启动时端口被占用？**
```bash
lsof -ti :8000 | xargs kill -9
lsof -ti :8501 | xargs kill -9
```

**Q：聊天返回 401？**
API Key 无效或未配置。检查 `.env` 文件。

**Q：PDF 上传无内容？**
需安装 `pip install pypdf`。

**Q：知识图谱不显示？**
确认后端已启动且 `data/syllabus.json` 存在。

**Q：如何更换主题？**
编辑 `frontend/.streamlit/config.toml` 和 `frontend/components/styles.py`。
